## 三、设计

**预览缩略图**: 每个模板显示业务类型对应的彩色图标 (📄合同/🧾发票/📋报销/💰费用)。
hover 时弹出 popover 显示名称、状态、code、版本、纸型、描述。

**卡片网格视图**: `grid-template-columns: repeat(auto-fill, minmax(220px, 1fr))` 自适应列数。
每张卡片有: 图标 + 状态标签 + 名称 + code + 类型 + 纸型 + 操作按钮。
左侧彩色边条标识业务类型。hover 时上浮 + 阴影。

**视图切换**: 筛选栏右侧的「📋 列表」/「▦ 卡片」按钮。

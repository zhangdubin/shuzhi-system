📖 使用说明

空状态：在表格/列表无数据时使用。提供 ico / t / m / actions 4 个槽位。

骨架屏：在数据加载时使用，避免布局抖动。提供 line / line.lg / line.sm / block / avatar 5 种规格。

全屏加载：路由切换/全局保存时使用。透明度 85% + 模糊背景。

JS 调用方式：

// 显示全屏加载 document.getElementById('loadingOverlay').classList.add('show'); // 隐藏 document.getElementById('loadingOverlay').classList.remove('show'); // 显示 Toast showToast('保存成功', 'success');
